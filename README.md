# Linearregression

Simple Ruby implementation of Linear Regression 

## Usage

`lr = Linear::Regression.new({1 => 2, 3 => 4, 5 => 6})`

*Hash keys match X values*

`lr.x`

*Hash values match Y values*

`lr.y`

*make projection*

`lr.projection(30)`
